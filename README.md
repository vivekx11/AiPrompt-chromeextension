# MAKE A PROPER PROMPT FOR YOUR CODE

Ye ek simple chat-based tool hai jo Google Gemini AI use karke aapki request ke basis par code generate karta hai.

## Features
- Natural language mein request karein (Hindi/English)
- AI automatically code generate karega
- Interactive chat interface
- Examples aur comments ke saath code

## Installation

1. Python install karein (3.7+)
2. Dependencies install karein:
```bash
pip install -r requirements.txt
```

## Usage

Tool run karein:
```bash
python gemini_chat_tool.py
```

## Examples

```
👤 Aap: mujhe ek calculator banana hai
🤖 AI: [Calculator code generate karega]

👤 Aap: ek todo list app banao
🤖 AI: [Todo list code generate karega]

👤 Aap: fibonacci series print karne ka function
🤖 AI: [Fibonacci function generate karega]
```

## API Key

Current API key already configured hai. Agar change karna ho to `gemini_chat_tool.py` mein `API_KEY` variable update karein.

API key working hai ya nahi, ye tool run karke check kar sakte hain!
