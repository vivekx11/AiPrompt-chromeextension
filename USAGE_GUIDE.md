# 📖 Usage Guide - AI Prompt Enhancer.

## 🎯 Kaise Use Karein

### Method 1: Direct Input Update (Recommended)

Ye method AI sites ke input fields ke liye best hai:

1. **AI Site Par Jao:**
   - ChatGPT (chat.openai.com)
   - Claude (claude.ai)
   - Gemini (gemini.google.com)
   - Ya koi bhi AI chat tool

2. **Prompt Likho:**
   ```
   mujhe calculator banana hai
   ```

3. **Text Select Karo:**
   - Mouse se drag karo
   - Ya Ctrl+A press karo (sab select)

4. **🤖 Button Click Karo:**
   - Button text select karne par appear hoga
   - Click karo

5. **Magic! ✨**
   - ⭕ Loading animation dikhega (2-3 seconds)
   - Enhanced prompt automatically input mein update hoga
   - ✅ Success notification dikhega

6. **Send Karo:**
   - Enhanced prompt ready hai
   - Enter press karo ya Send button click karo
   - Better AI response milega!

### Method 2: Modal View (Comparison)

Agar input field detect nahi hua:

1. Text select karo
2. 🤖 button click karo
3. Modal open hoga with:
   - 📝 Original Prompt
   - ✨ Enhanced Prompt
   - 📋 Copy button
4. Copy karke paste karo

## 💡 Pro Tips

### Tip 1: Select All
- Ctrl+A (Windows) ya Cmd+A (Mac)
- Pura prompt select ho jayega
- Ek click mein enhance!

### Tip 2: Partial Selection
- Sirf specific part select karo
- Wo part enhance hoga
- Baaki text same rahega

### Tip 3: Multiple Enhancements
- Ek prompt enhance karo
- Phir se select karke enhance karo
- Aur bhi better ho jayega!

### Tip 4: History Check
- Extension icon par click karo
- Recent enhancements dekho
- Previous prompts reuse karo

### Tip 5: Copy Enhanced Prompt
- Agar auto-update nahi hua
- Modal mein Copy button use karo
- Manual paste karo

## 🎯 Best Practices

### ✅ Do This:
- Clear, simple prompts likho
- Specific requirements mention karo
- Context provide karo
- Complete sentences use karo

### ❌ Avoid This:
- Bahut chote prompts (1-2 words)
- Unclear requirements
- Multiple unrelated requests ek saath
- Incomplete thoughts

## 🔧 Troubleshooting

### Issue: Button Nahi Dikha
**Solution:**
- Text properly select karo
- Extension enabled hai check karo
- Page refresh karo

### Issue: Input Update Nahi Hua
**Solution:**
- Modal se copy karo
- Manual paste karo
- Different input field try karo

### Issue: Loading Bahut Time Le Raha
**Solution:**
- Internet connection check karo
- API quota check karo
- Extension reload karo
